Password Strength Checker

A Python-based Password Strength Checker built using Tkinter and Regular Expressions (Regex).

Features

- Password strength analysis
- Uppercase and lowercase validation
- Number validation
- Special character validation
- Weak password detection
- Show/Hide password functionality
- User-friendly GUI using Tkinter

Technologies Used

- Python
- Tkinter
- Regular Expressions (Regex)
- Git & GitHub

How to Run

python password_strength_checker.py

Project Purpose

This project helps users create stronger passwords by checking security requirements and providing suggestions for improvement.
